# False Position

*This Function:*
* Finds the root of a function using the False Position method
* Calculates the Approximate Relative Error of the final solution
* Shows how many iterations were performed to solve for the root

*Required Inputs:*
* A function
* A lower guess of the root
* An upper guess of the root

*Optional Inputs:*
* The maximum relative error acceptable for an answer
* The maximum number of iterations to perform
